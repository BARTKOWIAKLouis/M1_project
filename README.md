## Get started :

### Nest API
Open a terminal and use these commands to launch the backend server
```bash
> cd nest-api
> npm install
> npm run start:dev
```

### React application
In another terminal use these commands to launch the frontend server
```bash
> cd react-app
> npm install
> npm run dev
```
